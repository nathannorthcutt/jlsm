## Summary

Fixed-byte-budget LRU block cache sitting in front of the sstable reader. Evicts by total bytes rather than by block count. Displaces the legacy striped-block-cache per sstable.byte-budget-block-cache.invalidates.

## Acceptance Criteria

- byte-budget eviction invariant test: cache never exceeds configured max bytes
- All requirements in sstable.byte-budget-block-cache direct-annotated
- Performance test confirming LRU under realistic query mixes
- Spec promoted DRAFT → APPROVED; striped-block-cache marked INVALIDATED per displacement chain
